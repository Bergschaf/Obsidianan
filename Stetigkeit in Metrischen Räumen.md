#definition  $f : X -> Y$ heißt stetig, in $a in X$, wenn
	für alle Folgen $x_k$ mit ${} lim_(n -> infinity) x_k = a$ gilt
$$lim_(n-> infinity) f(x_k) = f(a) $$


$f : X -> RR^n$ ist stetig, wenn $f_i : X -> RR$ (Komponentenfunktionen) alle stetig sind

### Alternative Charakterisierung
$f$ ist stetig in $a in X$, wenn 
$$forall epsilon > 0, exists delta > 0, forall x in X : quad d_X(x,a) <= delta => d_y (f(x), f(a))< epsilon$$
